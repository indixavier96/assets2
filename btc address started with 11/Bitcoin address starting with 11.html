You said:
A bitcoin address can be started with "11"?

ChatGPT said:

Yes — a Bitcoin address can start with "11", and it is valid.

Here’s why 👇

Why "11" is possible

Bitcoin legacy addresses (called P2PKH) use Base58Check encoding.

  • They always start with 1 on Bitcoin mainnet

  • The second character depends on the encoded payload + checksum

  • Because Base58 allows repeating characters, "11" can occur

So an address like:

Code
11XyZk4m5Pq8sL...

is perfectly valid.

Common Bitcoin address prefixes (for comparison)

Address typePrefix examples
Legacy (P2PKH)1... → can be 11..., 1A..., 1B..., etc.
SegWit (P2SH)3...
Native SegWit (Bech32)bc1...
Testnetm..., n..., tb1...

Important clarification

  • "11" does NOT mean:

    • special address

    • burn address

    • invalid address

  • It is simply a normal legacy Bitcoin address

The only way to know if an address is usable is whether:
✔ the checksum is valid
✔ someone controls the private key

Quick rule of thumb

Any address starting with 1 and passing checksum validation is valid — including 11….
